# 🎵 Sonosfera

Aplicação interativa que transforma som em visualização em tempo real.

A **Sonosfera** reage ao áudio captado pelo microfone, permitindo que músicos visualizem **intensidade**, **frequência** e **afinação** de forma intuitiva através de uma esfera dinâmica.

---

## ✨ Funcionalidades

- 🎤 Captura de áudio em tempo real  
- 🔵 Esfera que reage à intensidade (volume)  
- 🎯 Detecção de afinação (fica verde quando afinado)  
- 📊 Exibição da frequência em Hz  
- 🎶 Funciona com voz **e instrumentos musicais**  

---

## 🧠 Como funciona

A aplicação analisa o som captado pelo microfone e:

- Mede a **intensidade** → controla o tamanho da esfera  
- Detecta a **frequência** → exibe em Hz  
- Verifica a **afinação** → muda a cor da esfera  

---

## 🚀 Como usar

1. Abra o projeto no navegador  
2. Permita o uso do microfone  
3. Produza um som (voz ou instrumento)  
4. Observe a esfera reagindo em tempo real  

---
